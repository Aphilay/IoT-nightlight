# nightlight-demo
Nightlight demo using CounterFit

## Create a Python virtual environment
```
python3 -m venv .venv
```

## Install CounterFit pip packages:
```
pip install CounterFit
pip install counterfit-connection
pip install counterfit-shims-grove
```

## Run CounterFit:
```
CounterFit
```

## Run program:
```
python app.py
```
